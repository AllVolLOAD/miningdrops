# Майнинг с банковских карт

Статический веб-сайт для сервиса аренды банковских карт. Сайт предоставляет информацию о процессе аренды банковских карт, правовых документах и требованиях к участникам.

## Описание

Проект представляет собой одностраничный веб-сайт, разработанный для компании, которая предлагает услуги по аренде банковских карт для уменьшения комиссий при передаче средств. Сайт содержит информацию о процессе сделки, необходимых документах и требованиях к арендодателям.

## Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd miningdrops
```

2. Откройте `index.html` в браузере или используйте локальный сервер:

```bash
# С помощью Python
python -m http.server 8000

# С помощью Node.js (http-server)
npx http-server

# С помощью PHP
php -S localhost:8000
```

3. Откройте в браузере: `http://localhost:8000`

## Настройка

### Изменение видео фона
В файле `index.html` найдите строку с `data-property` и измените `videoURL`:

```html
<a id="bgndVideo" class="player"
  data-property="{videoURL:'YOUR_VIDEO_URL',...}">
</a>
```

### Изменение цветов и стилей
Основные стили находятся в:
- `css/style.css` - основные стили
- `scss/style.scss` - SCSS исходники (требует компиляции)

### Настройка формы
Форма обратной связи находится в секции `#contact-section`. Для подключения к серверу необходимо добавить обработчик на серверной стороне.
